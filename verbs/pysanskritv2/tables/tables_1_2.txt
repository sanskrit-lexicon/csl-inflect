; Additional conjugations not provided by Deshpande
; ---------------------------------------------------------------
; The conjugations of 'ru' are not written in Deshpande. However, he 
; presents mentions 'ru' in a sentence preceding 'stu'; the inference is that
; the conjugations of 'ru' are *like* those of 'stu'.  I.e., replacing
; 'st' in a conjugation of 'stu' with 'r' results in the corresponding
; conjugation of 'ru'.  There is only 1 exception the in the active voice
; conjugations of 'ru'; namely, in the 2,a,ipv 1p singular, the value for
; 'stu' is 'stuvAni', which would imply 'ruvAni';  but, there needs to be
; an 'nR' sandhi so the correct value is 'ruvARi'.
; currently, we don't show middle voice conjugations for 'ru'
